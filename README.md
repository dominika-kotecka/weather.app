# Aplikacja pobierająca dane pogodowe z miasta Kraków co 15 minut. 
### Języki :
- Python
- MySQL
### Biblioteki:
- requests
- schedule
- mysql-connector
### Instrukcja 
Utworzenie folderu venv
```python
py -m venv .venv
```
Instalacja bibliotek
```python
pip install -r requirements.txt
```
Aktywacja venv
```python
.venv/Scripts/activate
```
Odpalenie skryptu
```python
py main.py
```
